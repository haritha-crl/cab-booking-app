# CabNow - Advanced Cab Booking App

CabNow is a responsive single-page cab booking experience built with HTML, CSS, Bootstrap 5, and JavaScript. It includes a six-page booking flow with car selection, live tracking, payment, trip history, and rating.

## Features
- 8 car categories from Mini to Luxury
- Live tracking view with driver profile
- Dark and light mode toggle
- Promo code support with CAB20 discount
- SOS emergency button
- Payment, trip history, and rating flow

## Run Locally
1. Open the project folder in VS Code.
2. Launch [index.html](index.html) in your browser.
3. Or use a simple local server if you prefer.
